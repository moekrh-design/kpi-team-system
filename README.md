# نظام فريق - المهام ومؤشرات الأداء (نسخة HTML/Node)

هذه نسخة ويب خفيفة قابلة للتطوير، تدعم:
- تسجيل دخول (اسم مستخدم + كلمة مرور)
- إدارة مستخدمين (مدير/مشرف/موظف)
- مناسبات + مهام مرتبطة بالمناسبات
- تحديث إنجاز الموظف واعتماد المشرف
- مرفقات لكل مهمة (PDF/صور)
- تقارير بصيغة "خطاب رسمي" جاهزة للطباعة/PDF مع شعار الجهة من (الثوابت)
- تصدير التقارير إلى Excel (XLSX)
- تنبيه داخل النظام للمهام التي ينتهي وقتها خلال يومين
- نسخ احتياطي واستيراد نسخة (Backup/Restore) تشمل قاعدة البيانات + المرفقات
- وضع شاشة (Kiosk) يعرض المهام بالتناوب كل 5 ثواني مع مؤشرات الإنجاز (زر واحد للرجوع للنظام)

## التشغيل محليًا
1) ثبّت Node.js (إصدار 20 أو أحدث)
2) افتح المجلد ثم نفّذ:

```bash
npm install
npm start
```

ثم افتح:
http://localhost:3000

## المجلدات المهمة
- `uploads/` مرفقات المستخدمين + شعار/ختم الجهة
- `data/app.db` قاعدة بيانات SQLite

## النسخ الاحتياطي والاستيراد
من صفحة **الثوابت**:
- **حفظ نسخة احتياطية (ZIP)**: يحفظ قاعدة البيانات + مجلد المرفقات
- **استيراد نسخة**: يستبدل البيانات الحالية بالنسخة المرفوعة

> يتم حفظ نسخة تلقائية قبل الاستيراد (app.db.bak-... و uploads.bak-...).
